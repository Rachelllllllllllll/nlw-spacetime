## 💻 Projeto
Esse é um projeto Web responsivo de uma cápsula do tempo para exibir memórias em uma linha do tempo.

## 🚀 Tecnologias
Esse projeto foi desenvolvido durante o NLW da Rocktseat com as seguintes tecnologias:

- HTML
- CSS
- Git e Github
